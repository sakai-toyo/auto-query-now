Auto Query Now - 設定

使用例

設定例：

  • クエリパラメータ名: my_query
  • Enterキーを押す: チェック

この場合、URLが以下のようになっていると：

https://example.com/search?my_query=test

ページが読み込まれた時点でフォーカスされている入力欄（検索ボックスなど）に "test" が自動的に入力され、Enterキーが押されます。

使い方のヒント：

  • ページを開いた時に自動的にフォーカスされる入力欄（検索ボックスなど）で使用するのが便利です
  • 拡張機能は入力欄にフォーカスが当たるまで最大10秒間待機します
  • 複数のクエリパラメータを設定すると、順番に入力されます
  • Chrome DevToolsのConsoleで [Auto Query Now] のログを確認できます
  • React、Vue、Angularなどのフレームワークで作られたサイトにも対応しています
